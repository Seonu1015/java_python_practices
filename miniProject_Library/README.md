### 도서관 시스템 간단 구현

___

프로그래밍에 익숙해지기 위해 도서관 시스템을 간단하게 구현해 보았습니다.
교육 과정을 통해 배운 것을 바탕으로 python으로 먼저 작성하는 중이며,
완성된 후에는 java로 바꿔볼 예정입니다.

---

### _Python_


#### Main

- 회원 
  - 가입 및 로그인
  - 도서 검색 -> 찾는 책이 있으면 대여 요청, 없으면 구매 요청 가능
  - 자신이 신청한 목록 확인 가능
- 관리자(직원)
  - 최초 관리자 접속 시 등록 진행
  - 회원관리, 직원관리, 도서관리

#### Member

- 회원 
  - 회원 정보 등록
  - 중복 아이디 체크
- 관리자(직원)
  - 직원 아이디 및 비밀번호 자동 생성

#### book

- 도서 정보
- 도서관에 도서 등록
- 도서에 관한 대여여부 확인
